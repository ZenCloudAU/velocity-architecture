## Before You Contribute

This is a formally specified, trademarked framework. Contributions are welcome but the terms below apply to all submissions. By contributing, you agree to these terms.

---

## What Contributions Are Welcome

### Corrections and clarifications
- Typographical errors in any document
- Factual inaccuracies in the framework specification
- Broken links or outdated references
- Ambiguous wording that creates genuine confusion

### Examples and case studies
- Additional worked examples (Guardrail Canvas, Trade-off Matrix, ADR) demonstrating the framework in different contexts
- Domain-specific applications (government, healthcare, financial services, etc.)
- These must be clearly labelled as examples and not presented as modifications to the framework specification

### Translations
- Translations of the README and core templates into other languages
- Must include the trademark notice and attribution in the translated version

---

## What Is Not Accepted

- Changes to the framework specification (`spec/FRAMEWORK-SPEC.md`) without prior discussion with the owner
- Changes to the correspondence rules register that add, remove, or modify rules
- Additions of new viewpoints or instruments presented as official VAF components
- Anything that would alter the ISO/IEC/IEEE 42010:2022 conformance claim
- Commercial derivative works submitted as contributions

---

## How to Contribute

1. **Open an issue first** — describe what you want to change and why before submitting a pull request
2. **One change per pull request** — keep PRs focused and reviewable
3. **Follow the existing format** — templates use consistent structure; maintain it
4. **Attribution** — contributions will be acknowledged but copyright remains with Phil Myint / ZenCloud Global Consultants

## Code of Conduct

Contributions are professional exchanges. Keep feedback precise and constructive. Personal criticism, bad faith submissions, and attempts to circumvent the trademark or licence terms will result in the contribution being closed without comment.
