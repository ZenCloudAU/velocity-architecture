# 5.2 — The First Domain

The first domain is not chosen for visibility. It is chosen for leverage.

This article establishes how the first operational domain is selected inside a live organisation, how a single-threaded owner is named, and how the first compliant path is built.

Core concepts include:

- Decision Altitude
- Single-threaded ownership
- Compliant Paths
- The Tuesday Test
- Internal Market dynamics

The first functioning domain proves the decision system can survive contact with delivery reality.
