# Architecture Decisions

## Purpose

This directory is reserved for architecture decision records for Velocity Architecture.

## Naming Convention

Use ADR-YYYYMMDD-short-title.md.

## Status Model

Allowed statuses: Proposed, Accepted, Superseded, Rejected, Retired.

## Accountable Authority

Founder / CTO.

## No Fabricated Decisions

Do not create decisions that were not actually made and evidenced.
